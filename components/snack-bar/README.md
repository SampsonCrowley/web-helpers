# \<snack-bar>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i snack-bar
```

## Usage
```html
<script type="module">
  import 'snack-bar/snack-bar.js';
</script>

<snack-bar></snack-bar>
```

## Testing using karma (if applied by author)

```bash
npm run test
```
