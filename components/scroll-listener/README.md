# \<scroll-listener>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i scroll-listener
```

## Usage
```html
<script type="module">
  import 'scroll-listener/scroll-listener.js';
</script>

<scroll-listener></scroll-listener>
```
